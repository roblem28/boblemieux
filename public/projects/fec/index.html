Contributor ↔ Federal contract cross-reference. Pulls FEC contributions (Schedule A) for a set of contributor names + employer, scores each row's match confidence (0–100 from employer / occupation / state / name signals), and joins against USAspending.gov federal contract awards for the same recipient — so donations and contract dollars appear side by side. Defaults to the Fisher Sand & Gravel principals (ND/AZ), excluding the unrelated "Fisher Investments" / Ken Fisher. Name-only matches are flagged REVIEW.
