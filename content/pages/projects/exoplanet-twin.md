No tile servers exist for exoplanets — every surface is procedurally generated in real-time using GLSL shaders driven by real NASA confirmed planet data. Twelve confirmed exoplanets across eight type categories: hot Jupiters, lava worlds, ocean worlds, super-Earths, desert worlds, ice giants, gas giants, and mini-Neptunes. Each planet uses noise functions seeded from actual parameters — radius, temperature, host star type. Built with Three.js, zero external assets or API keys.
